# Uninstall rbenv
- Disable rbenv temporarily 
  - Zsh: remove `rbenv init` line
```sh
rbenv uninstall -f 2.7.5 # uninstall version (if ruby-build installed)
rm -rf `rbenv root` # uninstall completely
brew uninstall rbenv # remmove from brew
```
